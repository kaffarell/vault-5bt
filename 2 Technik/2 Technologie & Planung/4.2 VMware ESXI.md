
## Hypervisor
- Es gibt zwei arten von Hypervisor
	- Type 1 (bare metal)
		- Laufen direkt auf der Hardware
		- Sind statt einem Betriebssystem auf dem PC installiert
		- Beispiel: ESXI
	- Type 2 (hosted)
		- Laufen auf einem Betriebsystem
		- Beispiel Virtualbox auf Windows installiert
